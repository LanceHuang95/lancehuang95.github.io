1 绪论

1.数据结构

数据结构：相互之间存在一种或多种特定关系的数据元素的集合

根据视点的不同，数据结构分为逻辑结构和物理结构

逻辑结构：数据元素间的关系

  1. 集合结构
  2. 线性结构(数据元素一对一)
  3. 树形结构(数据元素一对多)
  4. 图形结构(数据元素多对多)

物理结构(存储结构)：数据的逻辑结构在计算机中的存储形式

  1. 顺序存储结构：数据元素放进地址连续的存储单元，其数据间逻辑关系和物理关系是一致的
  2. 链式存储结构：数据元素放进任意存储单元里，这组存储单元可以是连续的，也可以是不连续的

逻辑结构是面向问题的，物理结构面向计算机的

2.数据类型

数据类型：一组性质相同的值的集合
抽象数据类型ADT:自定义出来的集合

2 算法

算法：解决特定问题求解步骤的描述，在计算机中表现为指令的有限序列，并且每条指令表示一个或多个操作

1.算法的特性

  1. 输入输出
  2. 有穷性
  3. 确定性
  4. 可行性

    2.算法设计的要求

  5. 正确性

    算法程序无错误

算法程序对于合法输入数据能够产生满足要求的输出结果

算法程序对于非法输入数据能够产生满足规格说明的输出结果

算法程序对于精心选择甚至刁难的数据都有满足要求的输出结果
2. 可读性
3. 健壮性：输入数据不合法时，算法也能做出相关处理而不是产生异常或者崩溃
4. 时间空间效率

3.算法时间复杂度

算法时间复杂度：

  1. 大O记法：只保留最高阶
  2. 常数阶O(1) 线性阶O(n) 平方阶O(n^2) 对数阶O(log n)
  3. 最好最坏情况：无特殊说明，都是指最坏时间复杂度

O(log n)示例:

3. 线性表

线性表(List):零个或多个数据元素的有限 序列

1.线性表的抽象数据类型

ADT 线性表(List)
Data 
    线性表的数据对象集合为{a1,a2,...,an},每个数据元素类型均为Datatype，每个元素只有一个钱去和后继元素
Operation
    InitList(*L):初始化操作，建立一个空表L
    ListEmpty(L)
    ClearList(*L)
    GetElem(L,i,*e):将线性表L中第i个元素的值返回给e
    LocateElem(L,e):查找与e值相等的元素，若查找成功，返回该元素在表中的序列号；否则，返回0表示失败
    ListInsert(*L,i,e)
    ListDelete(*L,i,e)
    ListLength(L)
endADT

对于复杂的操作，都是由基本操作组成
以下实现两个线性表的并集

/* 将所有在线性表Lb中但不在La中的元素插入到La中 即La=La U Lb(并集)*/
void union(List * La，List Lb)
{
    int La_Len =ListLength(La);
    int Lb_Len =ListLength(Lb);
    ElemType e;
    for(int  i =1;i<Lb_Len;i++)
    {
        GetElem(Lb,i,e);
        if(!LocateElem(La,e))
            ListInsert(La,++La_Len,e);
    }

}

2.线性表的顺序存储结构

下面以一维数组来实现

/*线性表结构的定义*/
#define MaxSize 20
typedef int ElemType;
typedef struct 
{
    ElemType data[MaxSize];
    int length;
}SqList;

从上述代码所知，顺序存储结构需要三个属性：

  • 存储空间起始位置：数组data
  • 线性表最大存储容量：数组长度MaxSize
  • 线性表的当前长度：Length

假设线性表中每个元素占据c个存储单元(LOC表示获得存储位置的函数)，那么LOC(a i+1) =LOC(a i)+c，对于每个线性表位置的存入或取出数据所需时间相等，我们把具有这种特点的存储结构成为随机存储结构

获得元素操作：

int GetElem(SqList L,int i,ElemType *e)
{
    if(L.length ==0 || i<1 ||i>L.length)
        return false;
    *e =L.data(i-1);
    return true;
}

插入元素操作：

  • 判断插入位置，若位置不合理抛出异常

  • 如果线性表长度大于等于数组长度，抛出异常或者动态增加容量

  • 从最后一个元素向前遍历到第i个位置，分别将它们往后移一个位置

  • 将要插入的元素放入位置i处，表长+1

    /*操作结果：在顺序表L中第i个位置之前插入新的数据元素e */
int ListInsert(SqList *L,int i,ElemType e)
{
  if(L->length ==MaxSize) /*顺序表已满*/
      return false;
  if(i<1 || i>L->length+1)/*当i不在范围内*/
      return false;

  if(i<=L->length)/*插入数据位置不在表尾*/
  {
      for(int k=length-1;k>;k--)/*插入位置后数据元素向后移动一位*/
          L->data[k+1] =L->data[k];
  }

  L->data[i-1]=e; /*新元素插入*/
  L->length++;
  return true;
}

  • 删除元素操作*：

  • 判断位置，若不合理，抛出异常

  • 取出删除元素

  • 从删除元素位置开始遍历到最后一个元素，分别将其向前移动一个位置

  • 表长减1

    /*操作结果：删除顺序表L中第i个位置的元素，用e返回其值 */
int ListDelete(SqList *L,int i,ElemType *e)
{
  if(L->length ==0) /*线性表为空*/
      return false;
  if(i<1 || i>L->length) /*删除位置不正确*/
      return false;

  *e =L->data[i-1];/*取出元素*/
  if(i=L->length)/*删除的是顺序表的最后一个元素则将其置空*/
      L->data[i-1]=NULL;

  if(i<L->length) /*如果删除不是最后位置*/
  {
      for(int k=i;k<L->length;k++)/* 将删除位置后面的元素前移*/
          L->data[k-1] =L->data[k];
  }

  L->length--;
  return true;
}

    线性表的顺序结构在写入和读取时间复杂度O(1);插入和删除时时间复杂度O(n)

3.线性表的链式存储结构

定义链表结构

/*线性表的单链表存储结构*/
typedef struct Node /*定义节点类型*/
{
    ElemType data;
    struct Node *next;
}Node;
typedef struct LList  /*定义链表类型*/
{
    Node * header; /*头指针(属于头节点，指向第一个节点)(详见大话数据结构Page58 图3.6.5)*/
    int length;
}LList;

获得元素操作：

/* 用e返回L中第i个数据元素的值*/
/* 时间复杂度O(n)*/
int GetElem(LList L,int i,ElemType *e)
{
    if(L.length ==0 || i<1 ||i>L.length) /*位置异常*/
        return false;

    Node * pCur =L.header->next;/*定义pCur指向第一个节点*/
    int j=1;
    while(pCur && j<i) /*获取指向第i个节点的指针*/
    {
        pCur=pCur->next;
        ++j;
    }
    if(!pCur || j>i) /*第i个元素不存在*/
        return false;
    *e=pCur->data;
    return true;
}

单链表的插入

  1. 让待插入节点指向第i-1个节点的后继节点
  2. 让第i-1个节点指向待插入节点

注意：必须按顺序操作

/* 在L中第i个位置之前插入新的元素e*/
/*时间复杂度O(n)：需要遍历链表，单纯插入操作O(1)*/
int ListInsert(LList *L,int i,ElemType e)
{
    if(i<1 || i>L->length+1)/*当i不在范围内*/
        return false;    

    Node * pCur =L.header->next;/*定义pCur指向第一个节点*/
    int j=1;
    while(pCur && j<i-1) /*获取指向第i-1个节点的指针*/
    {
        pCur=pCur->next;
        ++j;
    }
    if(!pCur || j>i) /*第i个元素不存在*/
        return false;

    /*插入操作*/
    Node *s=(Node *)malloc(sizeof(Node));
    s->data=e;
    s.next=pCur->next;
    pCur->next=s;

    L->length++;
    return true;
}

单链表的删除
当前节点指向当前节点的后继节点的后继节点
即pCur->next =pCur->next->next

/*操作结果：删除单链表L中第i个位置的元素，用e返回其值 */
int ListDelete(LList *L,int i,ElemType *e)
{
    if(L.length ==0 || i<1 ||i>L.length) /*位置异常*/
        return false;

    Node * pCur =L.header->next;/*定义pCur指向第一个节点*/
    int j=1;
    while(pCur && j<i-1) /*获取指向第i-1个节点的指针*/
    {
        pCur=pCur->next;
        ++j;
    }
    if(!pCur || j>i) /*第i个元素不存在*/
        return false;

    /*删除第i个节点*/
    Node *pDel =pCur->next;
    pCur->next=pCur->next->next;
    *e=pDel->data;
    free(pDel);  /*释放该节点指向的内存，因为插入时malloc在堆上了*/

    L->length--;
    return true;
}

对于插入和删除越频繁的顺序表，单链表的效率越明显

单链表的创建
创建单链表的过程就是一个动态生产链表的过程，即从空表的初始状态起，依次建立各元素节点，并逐个插入链表

/*随机产生n个元素的值，建立带头结点的单链线性表L(头插法)*/
void  CreateLinkList(LList *L,int n)
{
    /*建立链表*/
    *L=(LList *)malloc(sizeof(LList));
    if (NULL == L)
    {
        return ;
    }

    L->header.data = NULL;
    L->header.next = NULL;
    L->length = 0;

    /*头插法插入节点*/
    srand(time(0));
    for(int i=0;i<n;i++)
    {
        /*创建新节点*/
        Node * new=(Node*)malloc(sizeof(Node));
        new.data=rand()%100+1;
        new.next=L->header.next;

        L->header.next=new;/*插入到表头*/
    }

    /*尾插法插入节点*/
    /*
    Node * rear =L->header.next;
    for(int i=0;i<n;i++)
    {
        Node * new=(Node*)malloc(sizeof(Node));
        new.data=rand()%100+1;
        rear->next=new;
        rear=new;
    }
    rear->next=NULL //不要忘记尾节点指向空
}

单链表的清空与删除
单链表的清空：删除每一个节点
单链表的删除：删除LList指针

void Clear_LinkList(LList * L)
{
    if(NULL == L)
        return;
    Node * pCurr=L->header.next;
    while(pCurr)
    {
        Node * pNext=pCurr->next;
        free(pCurr)
        pCurr=pNext;
    }
}

void Destroy_LinkList(LList * L)
{

    if (NULL == L)
    {
        return;
    }

    //清空链表
    Clear_LinkList(L);
    free(L);
    L = NULL;
}

4.静态链表

静态链表：用数组描述的链表,数组的元素都由两个数据域构成，data和cur(游标，相当于next指针,存放该元素的后继在数组中的下标)

  1. 为了方便插入数据，通常会把数组建立的大一些，以便有空闲空间
  2. 游标Cursor,为0表示无指向
  3. 数组的第一个和最后一个元素作为特殊元素，不存数据;通常把未被使用的数组元素成为备用链表(记住哪些地方为空，插入和删除时分配空间);数组的第一个元素的cur就存放备用链表的第一个节点的下标;数组的最后一个元素的cur则存放第一个有数值的元素的下标，相当于单链表的头节点(详见大话数据结构Page72 图3.12.1和3.12.2)
/*线性表的静态链表存储结构*/
#define MaxSize 1000
typedef struct StaticList
{
    ElemType data;
    int cur;  
} SList;

/*将一维数组space中各分量链成一备用链表*/
int InitStaticList(SList sapce)
{
    for(int i=0;i<MaxSize-1;i++)
    {
        space[i].cur=i+1;
    }
    sapce[MaxSize-1].cur=0; /*当前静态链表为空，最后一个元素的cur为0*/
    return true;
}

静态链表的插入

静态链表要解决的是：用静态模拟动态链表结构的存储空间的分配，需要时申请，无用时释放

  1. 静态链表操作的是数组，无法使用malloc和free,需要自己实现
  2. 实现malloc:从备用链表上去的第一个节点作为待插入的新节点
    /* 若备用空间链表非空，则返回分配的节点下标。否则返回0*/
int Malloc_StaticList(SList space)
{
 int i=sapce[0].cur;/*当前数组第一个元素的cur存的值就是要返回的第一个备用空闲的下标*/
 if(space[0].cur)
     sapce[0].cur=space[i].cur;
 return i;
}

/在静态链表L中第i个元素前插入新的数据元素e/
int ListInsert(SList L,int i,ElemType e)
{
if(i<1 ||i>ListLength(L)+1)
return false;
int k=MaxSize-1;//k为最后一个元素的下标
int j=Malloc_StaticList(L);//获得空闲的下标
if(j)
{
L.data=e;
for(int m=1;m<i;m++)
k=L[k].cur//找到第i-1个元素位置的下标
L[j].cur=L[k].cur; //把第i-1个元素的cur值(即下一个元素的下标)赋值给新元素的cur
L[k].cur=j; //把新元素的下标赋值给第i-1个元素的cur值

return false;
}

静态链表优缺点

  • 优点：在插入和删除时不需要删除元素，只需要移动游标，改进了顺序存储结构插入删除时需要移动大量数据元素的缺点
  • 缺点：没有解决连续存储分配带来的表长难以确定的问题;失去了顺序存储结构随机存取的特性

    5.循环链表

    循环链表：单链表中终端节点的指针由空指针改为指向头节点，使得单链表头尾相连(见大话数据结构Page79 图3.13.4)
  • 循环链表解决了从任意一个结点出发，访问链表全部节点的问题
  • 循环链表和单链表的差异：循环条件的判断上，判断p->next是否等于头指针
  • 在单链表中需要O(1)时间访问第一个节点，O(n)时间访问终端节点；在循环链表中，让指向终端节点的尾指针rear来指向循环链表,则查找终端节点O(1),查找第一个节点rear->next-next O(1)
    /*合并两个循环链表A B*/
p=rearA->next; /*保存A的头节点*/
rearA->next=rearB->next->next;/*A的终端节点指向B的第一个节点(不是B的头节点)*/
rearB->next=p;/B的终端节点指向A的头节点

    6.双向链表(double linked list)

    双向链表：在单链表的每个节点中都有两个指针，一个指向后继节点，另一个指向前驱节点。由于双向链表每个节点要记录两个指针，空间占用略多于单链表，属于空间换时间
    /*线性表的双向链表存储结构*/
typedef struct DoubleNode
{
  ElemType data;
  struct DoubleNode * prior;
  struct DoubleNode * next;
}DNode;
  • 双向链表的插入和删除*

先搞定待插入节点s的前驱和后继，再搞定p->next的前驱，最后解决p的后继

/*将s指向的节点插入节点p和p->next之间*/
s->prior=p;
s->next=p->next;
p->next->prior=s;
p->next=s;

/*删除节点p*/
p->prior->next=p->next;
p->next-prior=p->prior;
free(p);
